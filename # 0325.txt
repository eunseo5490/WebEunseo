# 0325

**제어문** : 코드의 흐름을 제어하는 문법

**조건식** : 연산의 결과가 true/false로 반환되는 식
  - 비교연산자, 논리연산자를 많이 사용

**조건문**(분기문) : 조건에 따라 흐름을 제어하는 문법
- **if문**(범위로 비교) : if, if-else, if-elseif
- **switch문**(값을 1ㄷ1로 비교) : 비교값과 일치하는 조건값의  case로 접근
  - ※ java12부터 switch문을 연산식으로 사용 가능


**반복문** : 조건에 따라 명령을 반복할지 말지 결정하는 문법
  - **for문** : 반복횟수가 정확하게 정해져있을 때 사용

  - **다중for문**
  - **while문** : 조건식이 참일 때 반복 (특정 조건을 달성하지 못하면 무한 반복해야 할 때 사용)
    - 반복횟수를 알 수 없을 때, 특정 조건을 달성하지 못하면 무한 반복해야 할 때 사용
  - **do-while문** : 
  - **기타제어문** : 내가 원하는 조건에 도달하면 반복횟수가 남았더라도 반복문을 끝내거나 다음 횟수로 건너뛰게 만들 수 있다.
    - break : 반복문 즉시 종료
