정규표현식(Regular Expression)
문자열의 "패턴'을 정의하는 규칙

0-9 : 0~9
가-힣 : 모든 한글

어노테이션을 직접 만드는 일은 거의 없다.
하지만 자바에서 제공하거나 라이브러리가 제공하는 어노테이션은 사용할 일이 많다.

메서드
- 구조
- 매개변수
- 반환타입, 반환값

중첩타입 파라미터(매개변수)

Outer<Inner<T>>
복잡한 데이터 구조 표현 가능
타입 안정성 유지
객체 관계를 명확하게 표현

제네릭 메서드
메서드 자체가 타입을 일반화해서 받는 메서드
클래스 전체를 제네릭으로 만드는게 아닌, 특정 메서드 하나만 필요할 때
타입을 유연하게 처리하는 방법

접근제한자 <T> 반환타임 메서드명(매개변수) {}

제네릭 타입 한정
제네릭 타입을 넣을 수 있는 곳이 너무 자유롭다보니 원하지 않는 것도 들어올 수 있음

<T extends 상위타입> : 상위타입을 상속하는 클래스들만 들어올 수 있음

제네릭 타입은 형변환이 안된다.
형변환을 성립하기 위해서는 와일카드문법을 이용해야 한다.
그렇기 때문에 매개변수 쪽에서 쓰인다.
<?>, <? extends T>, <? super T>
