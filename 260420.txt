입력 내용을 그대로 출력하는 명령어

파일 만들 때 사용하는 echo
echo. : 빈 내용
echo 내용 > 파일명
> : 덮어쓰기
>> : 추가쓰기

type : 파일에 들어있는 내용을 터미널에 출력하는 기능
- 로그 확인
- 설정파일 확인
- 간단한 텍스트 내용 확인
type 파일명1, 파일명2 : 여러개의 파일을 이어서 출력할 수 있다.

type 파일명1 > 파일명2 : 파일명1의 내용을 파일명2로 저장할 수 있다.

type 파일명1,파일명2 > 파일명3 : 두 파일을 하나로 합친다.

type con : 키보드에서 값을 입력받아 출력
ctrl + z 누르고 엔터치면 종료된다.

type con > 파일명 : 키보드에서 값을 입력받아 파일에 저장할 수 있다.
