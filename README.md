**제어문** : 코드의 흐름을 제어하는 문법

기본적인 코드의 흐름
위 -> 아래
좌 -> 우
(대입연산자 : 우 -> 좌)


**조건식** : 연산의 결과가 true/false로 반환되는 식
  - 비교연산자, 논리연산자를 많이 사용

**조건문**(분기문) : 조건에 따라 실행할 명령을 결정하는 문법
- **if문**(범위로 비교) : if, if-else, if-elseif
- **switch문**(값을 1ㄷ1로 비교) : 비교값과 일치하는 조건값의  case로 접근
  - ※ java12부터 switch문을 연산식으로 사용 가능


**반복문** : 조건에 따라 명령을 반복할지 말지 결정하는 문법
  - **for문**
