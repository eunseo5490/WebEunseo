
## 2차원 배열
1차원 배열을 요소로 갖는 배열

- 2차원 배열의 초기화
int [][] arr = { {1,2,3},{4,5,6},{7,8,9} };

- 2차원 배열의 선언과 생성
1차원 배열과 마찬가지로 길이를 지정해줘야 한다.
int [][] ar = new int[2][3]; //괄호2개 안에 숫자3개

- 배열의 각 공간에 접근하는 방법
만약 arr에 5에 접근해야 할 경우
System.out.println(arr[1][1]);

객체지향프로그래밍
ㄴ 객체들을 만들어서 프로그래밍을 하는 방식

객체 : 현실세계에서 표현할 수 있는 모든 것
유형의 물체, 무형의 것

현실세계의 객체를 소프트웨어 객체로 설계하는 것을 객체 모델링이라고 한다.

객체 모델링
객체의 특징과 동작을 추려내서 소프트웨어 객체의 필드와 메서드로 정의하는 과정
